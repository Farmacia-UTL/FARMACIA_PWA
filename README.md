# Proyecto1003 - WEB - PWA

La Plataforma Web Farmacéutica (PWA Farmacia) es una aplicación diseñada para digitalizar y optimizar los procesos de una farmacia, incluyendo la gestión de inventario, consulta de medicamentos, administración de proveedores, registro de ventas y control de pedidos. El sistema busca mejorar la eficiencia operativa, centralizar la información clave de la farmacia y ofrecer una experiencia moderna y accesible para usuarios y administradores desde cualquier dispositivo con acceso a Internet.

# Objetivo del proyecto

Desarrollar una plataforma web tipo PWA que permita gestionar de forma eficiente las operaciones esenciales de una farmacia mediante una interfaz moderna, segura y adaptable, ofreciendo acceso rápido a información de inventario, medicamentos, ventas y proveedores.
